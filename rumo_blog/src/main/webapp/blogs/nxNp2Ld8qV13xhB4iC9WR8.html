在线教育的混乱局面

发表于 2018/02/13 01:03:45   |   分类于 :  Java   |   访客 : 430   评论 : 0


随着信息技术迅速发展，特别是从互联网到移动互联网，创造了跨时空的生活、工作和学习方式，使知识获取的方式发生了根本变化。教与学可以不受时间、空间和地点条件的限制，知识获取渠道灵活与多样化。而资本市场和创业者的不断涌入，使得我国在线教育在近几年经历了一段快速发展期。

中国在线教育已进入智能教育时代

在线教育的发展与科技发展水平、教育理念变革以及用户教育需求升级和生活方式转变息息相关。随着互联网教育规模的不断扩大和商业模式的日渐稳定，教育用户学习需求的深化、消费意识的觉醒和消费能力的升级，中国在线教育在现阶段已经进入了学习领域垂直细分、学习方式丰富多样、资源开放共享、教育内容变现的智能教育时代。

智能教育时代，随着用户对知识需求深化和消费意识的觉醒，以“轻知识”为产品的知识付费平台相继出现。融合网校教育、MOOC、直播、知识付费等多种元素基础上，B2B2C在线教育平台成为互联网教育新模式。人工智能+教育成为新的技术研发方向，体现在各类教育产品中。

2018年即将进入到在线教育的第一个元年，直播和录播都不断的在抢占各自领域的是市场份额。以直播为首的腾讯课堂，录播为首的慕课网、51CTO、itbooking.net 等都不断地在刷新互联网在线教育的格局。

同时也在这个行业中出现了巨大的危机和隐患，课内容的泛滥和师资力量的不平衡以及不透明化是这个行业现在最大的问题，用户选择性过多抓住的用户群体也比较离散，直播课程大部分利用的是的休息时间或者周末时间来完成学习，但是这种不适合零基础用户的选择。

为什么不适合原因：

1：时间很难规划和安排。

2：人都是有惰性，都是一时的冲动型消费。

3：大部分的选择都是觉得自己需要改变，其实都是没有冷静和思考和对比结果。

4：大部分的在线教育直播课程都是晚上，大部分人都需要休息难以集中，效率也不高。

5：培训周期短，时效性不大，费用也相当昂贵，可能课程内容价值很高，比如以现在的IT编程为例，要学会一门语言的时间成本根本是不可能通过短短的三四个月就能学会更何况学习别人十几年拥有的内容，大部分也是规定好的内容，直接授课罢了，说白了就是徒有虚表。

6：大部分工作两三年的想提升，让别人抓住痛点，总以为通过几个月学习就能达到某个阶段，总想走所谓的捷径，这个世界上有很多的套路，就看谁的套路最深。

适合原因

1：有实际刚需，有工作经验者。

盈利模式

1：互联网上通过各种营销，广告，图片，拍照和文案来不停的刺激学习者购买相关课程和产品，而达到盈利。

2：大量的水军小号的配合，在线教育的课堂里面100人的课堂至少30%-60%的人都是水军。通过水军小号的刺激和配合完成交易。

3：强烈对比，强烈营销，骗取用户的信任。

建议用户选择

1：如果是初学者建议最好不要选择直播课程学习，自身先对这个行业进行了解和分析，以及前景分析，自身通过一个月到两个月的学习和观察在做选择。

2：通过小成本的录播课程的学习，比如在慕课网，itbooking.net中都可以找到很多。

3：执行学习计划和体系

4：然后找工作试错，在学习。

教育本身是一件伟大而又意义的事情，但是当今互联网在线教育还属于非常混乱的阶段。如果自己还没非常需要最好通过自学来完成所有自身的需求。每一个优秀的人都是通过自学学会的。然后在不断的跟着某一个阶段的老师来取长补短的过程。

在线教育的弊端

一项针对40000个社区的调查显示，相比传统的课堂教学，学生通过在线教育所获得的分数往往更低，他们也更容易退课。那么，在线教育的弊端具体有哪些呢？

1.学生与老师之间缺乏互动。

在传统的课堂里，老师提问学生回答，学生提出疑问，老师解答，全程可以交流。

在线授课一般通过某种平台，老师或是直播或是录播，老师与学生在授课过程中的沟通基本通过学生的留言老师的回复来达成。除授课过程中可能有所交流外，只能通过电子邮件或文本聊天进行交流。这样一来，学生与老师之间就缺乏应有的互动性。

2.学生与学生之间缺乏互动。

传统课堂中，老师提问，很多同学都有回答问题的机会，每个同学都可以听到其他同学的想法，便于对照自己的想法，看看自己缺在哪里。

而在线教育中，即使学生通过在线学习系统与导师互动，但在授课时仍然听不到其他学生的提问和讨论。学生只能接受无法回复，没有办法与他人交流，没有机会听取其他学生的问题和讨论。只听不说的教育模式是没办法提高学生的口语交际能力的。

3.学生与老师之间无法建立感情。

在传统的课堂里，老师传达知识的形式多种多样，板书、声音、眼神、手势、语气等，都可以成为传达知识的方式。学生与老师之间也就容易建立起感情。所以传统机构里学生有一种归属感，这是时间与老师积累的结果。

在线教育里面，学生只要求求知欲得到满足，才不管与老师之间有什么感情呢！这种无感情的学习方式是不容易收获成效的。

4.缺乏监督。

在传统的课堂里，班上少了哪一个同学没来，老师了如指掌。哪怕是到了大学，班上同学多了，老师无法记住每一个学生，但是也有点名机制可以保证学生们在规定的时间里整齐划一地出现在课堂里。

不少质疑在线教育的问题中，不约而同的提到自制力这个问题，没错，在线学习本身是一个自觉的过程。如果你不能管理时间和时间表，网上教育将不是一个适合你的学习方法。我上大学时有室友为了考六级在网上花“重金”买了某某老师的几十节课，由于没有人监督，一直到考试前十天她才想起来去看课，结果可想而知，要看的课程太多，时间太少，根本来不及。

5.授课形式单一。

教育是一门艺术，并非仅仅是交易那么简单。在传统的课堂里，老师与学生是能通过课堂与课堂之外的互动而建立感情的。老师授课的形式也不止于简单地口头讲述，还会有一些实验、小组活动、小型比赛等，这都能大大提高学生的口头表达能力与动脑动手的能力。

而在线教育无法实现这一点，在线授课模式是单一的老师讲课，学生听课，学生很难学到死知识之外的东西。

6.部分学生条件不足。

线上教育和线下教育有一个相同之处就是要收费。包括学费、书本费等。在偏远山区，学生们不要学费，很多甚至书本费也不要，这是个国家给予我们的特权。

而在线教育是要收学费的，有的时候，在线教育一节课的费用是比普通传统教育的收费高出很多的。即便是收费低的平台或教师，对于偏远山区的学生来说，仍然无法获得这种看似高级的听课模式。他们买一台电脑很贵，他们没有wifi,连网更是一件奢侈的事。

7.无法评价学生上课收益。

这就又和我们提到的互动有很大联系。在传统课堂中，学生可以通过课堂表现及时地反映出自己的听课效益。对于这一节课的内容，是完全吸收了，还是听懂了一大半，是听懂了一点点，还是完全不在状态，老师多多少少有个了解。

而在线授课的时候，老师看不到学生的表情，听不到学生说的话，不知道学生是以什么样的状态在听，甚至不知道他在没在听。这样是无法了解学生的听课收益并及时引导学生查漏补缺的。

为什么不选择在线教育，真相在这里

关于在线教育模式与传统授课方式，我们也只是给出参考，具体的好与不好，也要根据学生自身情况来判断。

不过，显而易见的是，传统的授课方式较之在线教育模式，还是“赢在起跑线上”的。古往今来，人们早已习惯学生们聚在一起，在老师的引导下，在周围同学们的互相帮助下，共同学习知识，共同进步。

难道不是吗？
