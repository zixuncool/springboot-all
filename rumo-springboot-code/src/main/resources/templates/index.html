二维码生成器

QRcode的方式来生成的，

  • 引入 QRcode.jar的包,墨风这个包，在mvnrepository不存在的。
  • 如何将maven仓库中不存在的jar，打入到仓库中，是我们要去学习的知识,

  • 如何安装第三方的jar包放入到maven仓库中：如下：

    前提条件：maven的环境变量要进行配置，并且执行mvn -version 能够得到maven的版本信息方可执行，否则无效
    mvn install:install-file -Dfile="D:\Qrcode.jar" -DgroupId=com.rumo.qrcode -DartifactId=Qrcode -Dversion=1.0 -Dpackaging=jar
  • 然后在项目的pom.xml文件中依赖安装好的qrcode：
    				
					com.rumo.qrcode
					Qrcode
					1.0
